Typora教程

Typora是一款简洁而强大的 Markdown 编辑器。它提供实时预览功能，让用户可以即时查看文档的渲染效果。

最初使用 Typora 是为了利用 markdown 写博客文章，但随着对于 Typora 的使用。发现 Typora 在日常笔记、记录过程相当好用，具体体现在：逻辑性较强的层次结构，功能多样的快捷键，简洁好看。

逻辑性体现在于 Typora 的大小标题、列表等元素上。这些大小标题、列表的组合可以迅速的构建起一个顺序、层次的文章结构。不但可以使得文章的思路清晰了然，也可以在写文章的时候锻炼写作的逻辑思维能力。

Typora 的几乎所有常用功能都可以使用快捷键实现，如：ctrl + 1~6 可以快速生成6种不同级别的标题，ctrl + shift + ~ 来生成一个代码行，ctrl + shift + k 来生成一个代码块。

学习下面教程之前需要修改Typora->文件->偏好设置中的部分设置：

标题：

一级标题：选中文字按 Ctrl + 1 或 # +文字

二级标题：Ctrl + 2 或 ## + 文字

三级标题：Ctrl + 3 或 ### + 文字

四级标题：Ctrl + 4 或 #### + 文字

五级标题：Ctrl + 5 或 ##### + 文字

六级标题：Ctrl + 6 或 ###### + 文字

一般来说，Typora的标题都是靠左的，想要居中的话需要利用 html 标签：

示例：

<center> 居中标题 </center>

居中标题

文字：

删除线：Alt + shift + 5 或 ~~文字~~

加粗：Ctrl + b 或 **文字**

斜体：Ctrl + i 或 *文字*

下划线：Ctrl + u

高亮：==文字== (博客上无法正常显示，但是 Typora 上能正常显示)

链接：

  1. 外部链接—超链接：

    用于跳转到其他页面的链接，格式：

    [提示文字](网址)

    快捷键：Ctrl + k

    示例：

    [百度](www.baidu.com)

    百度

  2. 内部链接—文章链接：

    用于跳转到文章的其他部分，格式：

    [提示文字](#标题)

    快捷键：Ctrl + k

    示例：

    [跳转到开头](#Typora教程)

    跳转到开头

  3. 自动链接：

    使用<>包裹任意 URL 或者邮箱，都会被转化为超链接

    示例：

    <http://www.baidu.com>

    http://www.baidu.com

上下标：

用 ^ 来生成上标，用 ~ 来生成小标：

示例：

O~2~

O~2~ ( 博客中无法显示，Typora中正常显示 )

V^3^

v^3^ ( Typora中正常显示 )

引用：

快捷键：Ctrl + shift + q 或 一级引用：>，二级引用：>>

使用 > 时后面需要跟空格，同时引用只有在一行的开头才能使用。

示例：
